Для добавления данного проекта в ваш проект - добавьте в конец .pro файла вашего проекта следующую строку: include('Путь к данному проекту в вашей системе'/toAdmin/toAdmin.pro)

Для использования C++ класса Admin подключите файл admin.h.

Перед сборкой в папке с данным проектом выполните SVN Update и обновите файлы данного проекта.

Note:
Admin admin(0,db);
admin.show();
где db - база данных.

В случае критических ошибок, связанных с запросами к базе данных, объект admin делает неактивными основные пункты таблиц и  посылает сигнал sError(QString) с текстом ошибок.
